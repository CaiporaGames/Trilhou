                            ### Simbolos deste documento ###

- ->: Este simbolo significa, próximo passo.


                            ### Abreviações ###

- BG: é uma abreviação para Background.

- BTN: é uma abreviação para botão.


                            ### Convenções ###

- Todos os scripts que começam com SO são scriptableObjects.

- Todos os scripts que começam com I são interfaces.

- Colocar namespace nos scripts.

- Nao usar variaveis ou funcoes publicas a menos que seja necessário. 

- Para mostrar os nomes das variaveis na janela do inspetor basta usar o atributo [SerializeField] variavelType variavelName;

- Usar Camel case para os nomes, ex: thisIsAnExample.

- Usar nomes descritivos nos metodos e nos nomes dos scripts.

- Faça commentarios nas variaveis e nos metodos.

- Use tooltips para mostrar no inspector as descricoes das variaveis.



                            ### Como usar Dialogue System ###

- Como configurar o dialogue na fase: Arraste o dialogueCnavas e o dialogueManger para a cena. Vá nos botões de NextBTN and BackBTN no canvas
        e no campo chamado OnClick arraste o dialogueManager para a caixa, se o botão é o NextBTN, onde aparece no function selecione a opção: DialogueManager -> 
        NextDialogue. Se o botão é o BackBTN, onde aparece no function selecione: DialogeManager -> BackDialogue

- Adicionar novos dialogos: Entre na pasta scriptableObject -> Click com o botão direito do mouse vá em: Create -> ScriptableObjects -> 
Dialogue. Um novo asset sera criado. Renome-o. -> Click no seu novo asset -> aperte o botao de +. Escreva o que desejar
       como dialogo. Coloque no nome do personagem na parte chamada names.

- Alterar ordem das falas no dialogo: Entre na pasta scriptableObject -> Click em dialogue -> segure com o botão esquerdo do mouse,
       nos 2 tracos do lado esquerdo da sentenca e arraste para a posição desejada.


                            ### Como usar o point bar ###

- Chame a funcão AddToPointsBar() para adicionar pontos na barra de points.


                            ### Como usar binoculos###

- No script the moveFOurDirections controle a velocidade e quanto do mapa o player pode olhar com o binoculo.
- No script binocular zoom tem que atribuir a figura a qual sera dado o zoom do binoculo.
- Esta faze tem a parte onde o player comeca sem estar usando o binoculo e depois ele vai para o binoculo. Controla-se isso com as duas 
       regioes que estao na hieraquia. So ativar/desativar o que precisar


                            ### Como usar roundeffect###
- Arraste o roundEfecctCanvas para a hieracquia
- Crie um manager e coloque o nextscenemanager script
- Crie dois pointos, um no meio do canvas e outro no lado esquerdo. Caso voce queira que o movimento comece
      pelo outro lado, entao coloque o segundo point na direita.
- No canvas, va em nextBTN e use a funcao nextLevel method that are on the nextscenemanager script
- No caso de fases que nao usem o index para passar voce tem que implementar o chamado dos metodos que estao 
      no nextSceneManager script
